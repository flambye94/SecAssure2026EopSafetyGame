# SecAssure2026EopSafetyGame
This repository contains the supplementary material and demo artefacts for the paper *A Safety-Aware Elevation of Privilege Game for ICS Threat Modeling Education: Experience from Three Workshop Iterations on Smart Grid Purdue Models Towards a Game Demo*.

The repository includes the Excel file containing the anonymised workshop data, facilitator coding tables, suggested Safety/HSE impact ratings, cyber-severity assessments, and notes from the three workshop iterations described in the paper. These materials support transparency and reproducibility by documenting how threats, affected devices or layers, STRIDE categories, cyber severity, and Safety/HSE impact were consolidated across the NIKT, UiA CTF, and UiA master-student sessions.

The repository also supports the development of a ready-to-run demo of the safety-aware Elevation of Privilege game. The game adapts Adam Shostack’s Elevation of Privilege threat-modeling card game to an Industrial Control Systems context by anchoring play in a Smart Grid Purdue-model architecture and adding an explicit Safety/HSE impact dimension to each identified threat.

The goal of the repository is to make the supporting data and workshop artefacts available for review, replication, teaching, and future evaluation of safety-aware threat modeling for IT/OT and critical-infrastructure cybersecurity education.
